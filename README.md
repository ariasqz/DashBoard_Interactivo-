# Dashboard de Clima y Noticias

Dashboard interactivo que muestra clima actual, pronóstico de 5 días y noticias filtradas por categoría.

## Tecnologías
- HTML5
- CSS3 (Flexbox, Grid, Responsive)
- JavaScript (ES6 Modules, Async/Await)
- APIs: OpenWeatherMap, GNews

## Características
- Búsqueda de clima por ciudad
- Pronóstico de 5 días
- Noticias con filtros de categoría
- Diseño responsive
- Animaciones suaves
